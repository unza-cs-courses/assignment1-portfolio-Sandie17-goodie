My Projects

1. Student Face Recognition Attendance System

An automated attendance system that uses facial recognition to identify students and record their attendance through a camera. The system detects and matches faces with stored student records and updates attendance in a database. It was developed using Python, OpenCV, machine learning algorithms, and MySQL with a simple HTML/CSS/JavaScript interface. The project solves the problem of time-consuming manual attendance and student impersonation, providing a faster and more accurate attendance process.

Live demo video

2. Career Guidance Mobile App for School Leavers

A web/mobile application designed to help school leavers choose appropriate career paths based on their interests, skills, and academic background. The system provides career information, recommendations, and guidance on required qualifications for different professions. It was built using HTML, CSS, JavaScript with a backend such as Node.js or Python and a MySQL/Firebase database. The application addresses the problem of limited access to structured career guidance, helping students make informed education and career decisions.

Live demo video

3. Hospital Patient Record Management System

A digital system developed to manage and store patient records in hospitals efficiently. The platform allows medical staff to register patients, update diagnoses, assign doctors, and track treatment history through a centralized database. The system was implemented using Java or Python, MySQL database, and web technologies such as HTML, CSS, and JavaScript. It solves the challenge of paper-based record management by improving data accessibility, accuracy, and overall hospital operational efficiency.
